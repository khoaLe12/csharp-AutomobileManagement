# Automobile Management Web
This is a simple web used to perform some basic CRUD actions

## Tech Stack
* IDE: Visual Studio
* Frameworks: .NET 6, EF Core, ASP.NET Core Web App (MVC)
* Database: MS SQL Server
* Query tech: LINQ
